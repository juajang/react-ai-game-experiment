# 🐔 닭 농장 어드벤처

React로 만든 닭 키우기 시뮬레이션 게임입니다. 닭을 키우고, 농장을 확장하며, 모험을 떠나세요!

## ✨ 주요 기능

### 🐣 닭 키우기 시스템
- **성장 단계**: 알 🥚 → 병아리 🐥 → 청소년 🐤 → 성체 🐔
- **스탯 관리**: 배고픔, 행복도, 건강, 피로도
- **알 낳기**: 건강한 닭은 정기적으로 알을 낳습니다
- **레벨 시스템**: 모험을 통해 경험치를 얻고 레벨업

### 🏗️ 농장 건설
- **닭집 🏠**: 닭이 쉬고 회복할 수 있는 공간 (35💰)
- **연못 💧**: 주변 닭의 건강 증진 (25💰)
- **꽃/꽃덤불 🌸**: 닭의 행복도 상승 (10💰/20💰)
- **풍차 🌀**: 모든 스탯 향상 - 황금 농장 전용 (100💰)
- **우주선 🚀**: 엔드게임 컨텐츠 (500💰 + 특수 재료)

### 🗺️ 모험 시스템
- **월드맵 탐험**: 성체 닭을 모험에 보낼 수 있습니다
- **자원 수집**: 물, 벼를 소모하며 새로운 타일 탐험
- **경험치 획득**: 이동하면서 경험치를 얻고 레벨업
- **특수 아이템**: 모험에서 삽, 금속 조각 등 발견

### 🎯 농장 등급
- 🐥 **병아리 농장** (0-4마리)
- 🐔 **닭 농장** (5-9마리)
- ✨ **황금 닭 농장** (10마리 이상)

## 🚀 시작하기

### 필수 요구사항
- Node.js (v16 이상)
- npm 또는 yarn

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프리뷰
npm run preview
```

## 🎮 게임 방법

### 기본 조작
- **클릭**: 필드를 클릭하여 벼(사료) 배치
- **드래그**: 닭을 드래그하여 이동
- **건물 이동**: 건물을 드래그하여 위치 변경

### 아이템 패널 (좌측)
- 🌾 **벼**: 닭이 먹는 사료 (3💰)
- 🌸 **꽃**: 행복도 증진 (10💰)
- 🌺 **꽃덤불**: 강력한 행복도 효과 (20💰)
- 💧 **연못**: 건강 증진 (25💰)
- 🏠 **닭집**: 휴식 공간 (35💰)
- 🌀 **풍차**: 전체 스탯 향상 (100💰)
- 🚀 **우주선**: 특수 재료 필요 (500💰)

### 스탯 관리
- **배고픔**: 벼를 먹으면 회복
- **행복도**: 꽃 근처에서 증가, 똥이 있으면 감소
- **건강**: 연못, 닭집에서 회복
- **피로도**: 닭집에서 잠들어 회복

### 모험 보내기
1. 성체 닭의 상태바에서 **🗺️ 모험** 버튼 클릭
2. 주사위를 굴려 이동
3. 물과 벼를 관리하며 탐험
4. 피로도가 100%가 되거나 자원이 떨어지면 귀환

### 팁
- 닭집을 지어 닭들이 쉴 수 있게 하세요
- 똥은 클릭하여 치우거나 모험에서 얻은 삽으로 제거하세요
- 행복도 80% 이상일 때만 레벨업이 가능합니다
- 황금 농장을 달성하면 계속 플레이할 수 있습니다
