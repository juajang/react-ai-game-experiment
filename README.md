# 🐔 치킨 시뮬레이터

React로 만든 닭 키우기 시뮬레이션 게임입니다. 닭을 키우고, 농장을 확장하며, 모험을 떠나 황금 닭 농장을 달성하세요!

## ✨ 주요 기능

### 🐣 닭 키우기 시스템

- **성장 단계**: 알 🥚 → 병아리 🐥 → 청소년 🐤 → 성체 🐔
  - 알: 따뜻함을 받으면 부화 (120초)
  - 병아리: 120초 후 청소년으로 성장
  - 청소년: 150초 후 성체로 성장
- **스탯 관리**: 배고픔, 행복도, 건강, 피로도
  - 배고픔: 벼를 먹으면 회복 (30 포인트)
  - 행복도: 꽃/꽃덤불 근처에서 증가, 똥이 있으면 감소
  - 건강: 연못, 닭집에서 회복
  - 피로도: 닭집에서 잠들어 회복
- **알 낳기**: 건강한 닭(행복도 80% 이상, 건강 50% 이상, 배고픔 40% 이상)은 정기적으로 알을 낳습니다
- **레벨 시스템**: 모험을 통해 경험치를 얻고 레벨업 (레벨 1부터 시작, 경험치 필요량은 1.5배씩 증가)
- **이름 변경**: 닭의 이름을 자유롭게 변경할 수 있습니다

### 🏗️ 농장 건설

#### 기본 건물
- **벼 🌾**: 닭이 먹는 사료 (3💰)
- **꽃 🌸**: 행복도 증진 (10💰, 효과 범위 70px)
- **연못 💧**: 주변 닭의 건강 및 행복도 증진 (100💰, 효과 범위 80px)
- **닭집 🏠**: 닭이 쉬고 회복할 수 있는 공간 (100💰, 수용 인원 4마리)

#### 황금 농장 전용 건물 (10마리 이상)
- **꽃덤불 🌺**: 강력한 행복도 효과 (300💰, 효과 범위 55px)
- **풍차 🌀**: 모든 스탯 소폭 향상 (300💰, 효과 범위 100px)

#### 특수 건물 (재료 필요)
- **과학기지 🔬**: 특수 아이템 제작 가능 (500💰 + 나뭇가지 더미 3개 + 조개껍데기 3개)
- **자동사료 배분기 🤖**: 주기적으로 사료를 자동 배분 (500💰 + 금속조각 1개 + 나뭇가지 더미 2개)
- **우주선 🚀**: 엔드게임 컨텐츠 (1000💰 + 우주선 플레이트 1개 + 설계도 조각 1개 + 연료 전지 1개)
- **닭의 성 🏰**: 엔드게임 컨텐츠 (1000💰 + 비틀어진 덩굴줄기 1개 + 나뭇가지 더미 3개)

### 🗺️ 모험 시스템

성체 닭을 모험에 보내 월드맵을 탐험할 수 있습니다.

#### 모험 시작 조건
- 성체 닭만 모험 가능
- 피로도 100% 미만
- 잠자는 상태가 아님

#### 모험 메커니즘
- **주사위 시스템**: 레벨에 따라 라운드당 주사위 횟수가 결정됩니다
  - 레벨 1: 1회
  - 레벨 2: 2회
  - 레벨 3: 3회
  - 레벨 4: 4회
  - 레벨 5: 5회
- **자원 관리**: 물과 벼를 소모하며 탐험
  - 물: 레벨에 따라 15~35 제공 (레벨 1: 15, 레벨 5: 35)
  - 벼: 레벨에 따라 5~13 제공 (레벨 1: 5, 레벨 5: 13)
  - 이동 시 물 1 소모
  - 조사 시 벼 1 소모
- **피로도**: 주사위를 굴릴 때마다 피로도가 증가 (레벨이 높을수록 증가량 감소)
- **경험치 획득**: 이동할 때마다 5 EXP 획득

#### 모험 종료 조건
- 피로도 100% 도달
- 물이 다 떨어짐
- 벼가 다 떨어짐
- 사망 (특수 이벤트)
- 수동 귀환

#### 월드맵 특징
- **Fog of War**: 탐험하지 않은 지역은 가려져 있습니다
- **다양한 지형**: 초원, 숲, 해변, 산, 바다, 길 등
- **특수 장소 (POI)**:
  - 🏚️ 버려진 민가: 삽, 나뭇가지 더미 획득 가능
  - 🚀 발사장: 우주선 설계도 조각 100% 획득
  - 📡 통신탑: 부서진 안테나 100% 획득
  - 🏭 폐공장: 금속 조각 100% 획득
  - 🔬 버려진 과학기지: 연료 전지 100% 획득
  - 기타: 전초기지, 농장 등

#### 모험에서 획득 가능한 아이템
- **삽 🪏**: 똥을 제거하는 도구 (버려진 민가, 초원, 숲, 해변, 마을에서 획득)
- **나뭇가지 더미 🪵**: 건물 건설 재료 (초원, 숲, 버려진 민가에서 획득)
- **비틀어진 덩굴줄기 🌿**: 닭의 성 건설 재료 (숲에서 획득)
- **조개껍데기 🐚**: 과학기지 건설 재료 (해변에서 획득)
- **금속 조각 ⚙️**: 자동사료 배분기 건설 재료 (폐공장, 자원 포인트에서 획득)
- **우주선 설계도 조각 📜**: 우주선 건설 재료 (발사장에서 100% 획득)
- **부서진 안테나 📡**: 과학기지 제작 재료 (통신탑에서 100% 획득)
- **연료 전지 🔋**: 우주선 건설 재료 (버려진 과학기지에서 100% 획득)

### 🎯 농장 등급

- 🐥 **병아리 농장** (0-4마리): 기본 등급
- 🐔 **닭 농장** (5-9마리): 중급 등급
- ✨ **황금 닭 농장** (10마리 이상): 최고 등급 (풍차, 꽃덤불 건설 가능)

### 💰 경제 시스템

- 닭이 행복하게 생활하면 코인을 획득합니다 (행복도 30% 이상일 때)
- 성체: 2코인, 청소년: 1코인, 병아리: 0코인
- 획득 확률: 20%

### 🎮 게임 상태

- **게임 오버**: 모든 닭이 사망
- **게임 클리어**: 황금 닭 농장 달성 (10마리 이상)
- 클리어 후에도 계속 플레이 가능

## 🚀 시작하기

### 필수 요구사항

- Node.js (v16 이상)


## 🎮 게임 방법

### 기본 조작

- **클릭**: 필드를 클릭하여 선택한 아이템 배치
- **드래그**: 닭을 드래그하여 이동
- **건물 이동**: 건물을 드래그하여 위치 변경 (닭집, 연못, 꽃덤불, 풍차, 과학기지, 자동사료 배분기, 닭의 성)
- **똥 제거**: 똥을 클릭하거나 삽 도구로 제거

### 아이템 패널 (좌측)

좌측 패널에서 아이템을 선택하여 배치할 수 있습니다.

#### 소비 아이템
- 🌾 **벼**: 닭이 먹는 사료 (3💰)

#### 장식 아이템
- 🌸 **꽃**: 행복도 증진 (10💰)

#### 건물
- 💧 **연못**: 건강 및 행복도 증진 (100💰)
- 🏠 **닭집**: 휴식 공간 (100💰)

#### 황금 농장 전용
- 🌺 **꽃덤불**: 강력한 행복도 효과 (300💰)
- 🌀 **풍차**: 전체 스탯 향상 (300💰)

#### 특수 건물 (재료 필요)
- 🔬 **과학기지**: 특수 아이템 제작 (500💰 + 재료)
- 🤖 **자동사료 배분기**: 자동 사료 배분 (500💰 + 재료)
- 🚀 **우주선**: 엔드게임 컨텐츠 (1000💰 + 재료)
- 🏰 **닭의 성**: 엔드게임 컨텐츠 (1000💰 + 재료)

### 스탯 관리

- **배고픔**: 벼를 먹으면 회복 (30 포인트)
- **행복도**: 
  - 꽃 근처에서 증가 (0.1)
  - 꽃덤불 근처에서 증가 (0.15)
  - 연못 근처에서 증가 (0.5)
  - 똥이 있으면 감소 (0.15)
- **건강**: 
  - 연못 근처에서 회복 (0.2)
  - 닭집에서 회복 (1.0)
  - 배고프거나 피곤하거나 불행하면 감소
- **피로도**: 
  - 닭집에서 잠들어 회복 (3.0)
  - 활동 시 증가 (0.3)

### 모험 보내기

1. 성체 닭의 상태바에서 **🗺️ 모험** 버튼 클릭
2. 우측 모험 패널에서 주사위를 굴려 이동
3. 물과 벼를 관리하며 탐험
4. 타일을 조사하여 아이템 획득
5. 피로도가 100%가 되거나 자원이 떨어지면 자동 귀환
6. 귀환 시 경험치 획득 및 레벨업 가능

### 도구 사용

- **삽 🪏**: 인벤토리에 삽이 있으면 도구 패널에서 선택 가능
- 삽을 선택하면 마우스 커서가 삽으로 변경
- 똥 위에 마우스를 올리고 클릭하면 제거 (삽은 소모되지 않음)

- 황금 농장을 달성하면 풍차와 꽃덤불을 건설할 수 있습니다
